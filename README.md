# graphql-project
入门graphql的小实践

## 项目使用技术栈
```javascript
server: 
koa + graphql + mongodb

client:
react hooks + ts
```

## 安装项目依赖

```javascript
server: 
yarn add koa koa-router koa2-cors graphql koa-graphql mongoose

client:
npx create-react-app client --typescript
yarn add antd less less-loader
```
